# MonacoMed – Launch-Checkliste (SEO & Technik)

Diese Punkte gehören NICHT in den Design-Entwurf, sondern werden beim Live-Gang umgesetzt:

## Muss vor Launch
- [ ] Impressum & Datenschutz: Platzhalter durch geprüfte Texte ersetzen (Anwalt/Generator)
- [ ] Cookie-/Consent-Banner nur falls Analytics o. Ä. eingesetzt wird
- [ ] Formulare an echtes Backend/E-Mail anbinden (inkl. DSGVO-Checkbox)
- [ ] Sprechende URLs: /arbeitsmedizin, /bahnmedizin, /betriebsarzt-muenchen …
- [ ] 404-Seite

## SEO-Grundausstattung
- [ ] Canonical-Tags auf allen Seiten
- [ ] Open-Graph- + Twitter-Card-Tags (Titel, Beschreibung, Vorschaubild)
- [ ] sitemap.xml + robots.txt, Anmeldung Google Search Console
- [ ] Strukturierte Daten: LocalBusiness/Physician (Betriebsarzt München), Organization (alle Seiten), BreadcrumbList (Leistungsseiten), FAQPage (FAQ)
- [ ] NAP-Konsistenz (Name, Adresse, Telefon) + Google Business Profile

## Performance
- [ ] Fonts lokal hosten (DSGVO) + preload, WOFF2-Subset
- [ ] Bilder als WebP/AVIF mit width/height-Attributen, Logos als SVG wo möglich
- [ ] Lighthouse-Check (Ziel: 90+ mobil)
